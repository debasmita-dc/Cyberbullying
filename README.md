# Cyberbullying
📌 Project Overview

This project focuses on detecting cyberbullying in online text using Natural Language Processing (NLP) and Machine Learning techniques. The system analyzes user-generated content and classifies it based on toxicity, emotion, and threat level, helping identify harmful or abusive language on digital platforms.

🎯 Objectives

Detect cyberbullying and toxic content in text data
Classify severity levels (Low / Medium / High)
Map detected content to relevant legal provisions
Provide a simple web-based interface for analysis and reporting

🛠️ Technologies Used

Python

Flask (Web Framework)

NLP & Machine Learning (BERT-based model)

SQLite (Database)

HTML, CSS, Bootstrap

Chart.js (Dashboard Visualization)

⚙️ System Architecture

User submits text through the web interface
Text preprocessing (emoji normalization, slang handling, profanity masking)
ML model predicts toxicity, emotion, and threat
Severity level is determined
Results are stored and displayed on the dashboard

🚀 Features

Real-time cyberbullying detection
Severity classification (Low / Medium / High)
Legal mapping to Indian laws (IPC & IT Act)
Complaint storage and analytics dashboard
User-friendly interface

📊 Output

Classified text with severity level
Dashboard showing complaint statistics
Stored records for further analysis

📌 Use Case

This system can be used by:
Educational institutions
Social media moderation teams
Cybercrime awareness and research projects
